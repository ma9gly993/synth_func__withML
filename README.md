Описание проекта
Данный проект был создан для участия в альтернативном экзамене по дисциплине МЛиТА.
Тема: Нейросетевые методы аппроксимации булевых функций.

Сравнение моделей машинного обучения (MLP, Decision Tree, Logistic Regression) на синтетических булевых датасетах с автоматическим подбором гиперпараметров через Optuna и анализ метрик (Accuracy, F1, ROC-AUC) при дисбалансе классов 90/10.

Для работы выбрано 3 основных вида датасетов:

Monk's problems и созданные вариации (MONK-1, MONK-2, MONK-3)

Искусственно созданная функция, содержащая операторы AND, XOR, NOT

Зашифрованные данные (битовый шифр c=(x+K+2r) mod N)

Примечание: папки data_old, results_old содержат результаты предыдущих экспериментов. Для текущей работы использован новый датасет mod3count.

Для воспроизведения результатов: поместите датасеты в /data, запустите main.py. Создастся папка /results с файлом metrics.csv (подобранные гиперпараметры + все метрики).
